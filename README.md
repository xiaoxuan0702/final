# 學生社團管理系統（Student Club Management System）

## 專題簡介
本專題旨在設計並規劃一套學生社團管理系統，提供學校與學生一個
集中管理社團、社員與活動的線上平台，以改善傳統紙本與人工管理效率低落的問題。

## 系統角色
- 一般學生
- 社團幹部
- 系統管理員

## 系統功能
- 使用者註冊與登入
- 社團資訊瀏覽與管理
- 社員申請與審核
- 社團活動建立與報名
- 公告發布與通知

## 使用技術
- 前端：HTML / CSS / JavaScript
- 後端：系統設計階段（可擴充）
- 資料庫：MySQL

## 專題目標
- 練習系統分析與設計流程
- 建立完整系統文件
- 熟悉 GitHub 專案管理
